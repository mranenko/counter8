# counter8

fintech/healthtech project built with the following:

- Next JS
- React
- Tailwind
- TypeScript

node + npm

- node 18.15.0
- npm 9.5.0


## project creation

```bash
  npm install next react react-dom
  npm install --save-dev typescript @types/react @types/node
  npm install --save-dev tailwindcss postcss autoprefixer
```


## run

```bash
  npm install
  npm run dev
```
